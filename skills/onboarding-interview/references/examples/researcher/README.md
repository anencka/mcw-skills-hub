# Example portfolio — a biomedical researcher / technology developer

A **filled-in reference** showing what a good `~/work-profile/` looks like. The persona (Dr. Maya
Okonkwo) is fictional and general — a translational researcher who both runs studies and builds
technology — so it reads as an example, not a template to copy. Use it to show the operator what
"good, specific, one-page" looks like. **Never copy these answers into a real operator's files.**

Included here: `identity`, `role-and-responsibilities`, `current-projects`, `communication-style`,
`domain-knowledge` (the highest-value files). The other five follow the same shape.
